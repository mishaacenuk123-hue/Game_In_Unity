------------------------Інструкція запуску проекта(Тимчасово)------------------------  
1.Отримайте файли проекту:  
*Якщо це ZIP-архів, розпакуйте його в зручну для вас папку.  
*Якщо це Git-репозиторій, клонуйте його на свій комп'ютер.  
2.Відкрийте Unity Hub:  
*Запустіть Unity Hub (це менеджер проектів Unity).  
3.Відкрийте проект:  
*Натисніть кнопку "Open" (Відкрити) в Unity Hub.  
*З'явиться вікно для вибору папки. Знайдіть і виберіть основну папку проекту (ту, де знаходиться папка Assets, ProjectSettings тощо).  
*Натисніть "Select Folder" (Вибрати папку).  
4.Очікуйте імпорту:  
*Unity може почати імпортувати ассети та конвертувати проект під вашу версію редактора. Це може зайняти деякий час.  
(Бажано відкривати на Unity 6000.0.59f2)  
5.Запустіть проект:  
*Після завантаження проекту в редакторі, натисніть велику кнопку "Play" (Трикутник) вгорі по центру редактора, щоб запустити гру/додаток.
