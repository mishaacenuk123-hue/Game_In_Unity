1.Переконайтесь що у вас заванатажений Unity Hub та Unity.  
2.Перейдіть у папку "Game unity" потім в папку "Assets" тоді в "Levels" і відкрийте файл "Level1.unity".  
3.Проект відкриється в Unity.  
4.Зверху над сценою нажати кнопку "▶".
